# Репозиторий проекта "Влияние погодных условий на ДТП"

## Цель проекта
Исследовать влияние погодных условий на количество и тяжесть ДТП на примере четырёх городов России:
 - Краснодар
 - Сочи
 - Новороссийск
 - Армавир

## Технологии
 - Python (ETL)
 - Supabase (PostgreSQL, SQL), была создана база в декларативном стиле
 - DataLens (визуализация)

## Порядок выполнения
  1. Парсинг городов с Wikipedia
  2. Обработка координат
  3. Получение кодов ГИБДД
  4. Загрузка погоды
  5. Загрузка ДТП

## Скрипт запускается в командной строке конструкцией python my_dtp_script
